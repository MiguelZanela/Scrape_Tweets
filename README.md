# Scraping tweets

## Source
Based on the lib [snscrape](https://pypi.org/project/scrape/) maintained by [JustotherArchivist](https://github.com/JustAnotherArchivist) having all the documentation and code in that [repository](https://github.com/JustAnotherArchivist/snscrape)

## This code
This solution focuses on searching for specific hashtags within a period of time on the tweeter. It is possible to modify this solution to search for words or phrases and users. Still, for that, it is necessary to see the original documentation to make these changes.
